# Simple_Resume

Simple_Resume 是一个纯前端在线简历编辑器，不依赖数据库和服务器，打开网页即可编辑、预览并导出简历。

## 项目特点

- 三栏工作台布局：左侧设置、中间编辑、右侧预览
- 支持基础信息、经历模块、文本模块
- 支持模块新增、删除、显示隐藏、拖拽排序
- 支持头像上传和自定义信息项
- 支持三种排版风格切换
- 支持背景色、行高、页边距、模块间距调节
- 支持分层字体控制
- 支持本地自动保存
- 支持撤销 / 重做
- 支持中英文切换
- 支持智能分页与 PDF 导出
