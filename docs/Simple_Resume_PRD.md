# Simple_Resume 产品需求文档（PRD）

## 1. 产品概述

### 1.1 产品名称
Simple_Resume

### 1.2 产品定位
Simple_Resume 是一个纯前端在线简历编辑器，用户无需注册、无需数据库、无需服务器，即可在浏览器中完成简历编辑、排版调整、实时预览和 PDF 导出。

### 1.3 产品目标
- 提供低门槛、开箱即用的简历编辑体验
- 支持模块化管理和灵活排版
- 让预览效果尽量接近最终导出效果
- 在纯前端前提下实现较完整的编辑器能力

## 2. 目标用户

- 应届生和校招生
- 需要快速制作简历的求职者
- 希望通过网页直接完成简历排版的用户
- 不希望依赖后端服务、账号系统的个人用户

## 3. 用户核心需求

- 可以方便编辑简历内容
- 可以自由增删模块并调整顺序
- 可以实时查看简历成品效果
- 可以灵活调整字体、字号、边距、行距等排版项
- 可以导出正式的 PDF 简历
- 可以减少分页断裂、留白过多等导出问题

## 4. 产品结构

Simple_Resume 采用三栏工作台布局：

- 左侧：排版设置 + 模块管理
- 中间：模块编辑区
- 右侧：A4 简历实时预览区

顶部提供全局操作区，包括撤销、重做、语言切换和导出。

## 5. 功能清单

### 5.1 模块管理
- 支持基础信息模块
- 支持经历类模块
- 支持文本类模块
- 支持新增模块
- 支持删除模块
- 支持模块显示 / 隐藏
- 支持模块拖拽排序
- 支持点击模块切换当前编辑对象

### 5.2 内容编辑

#### 基础信息模块
- 编辑姓名
- 上传照片
- 删除照片
- 编辑求职意向 / 个人总结
- 动态添加自定义信息项
- 为信息项配置图标
- 编辑信息项内容
- 删除信息项

#### 经历类模块
- 编辑模块标题
- 新增经历条目
- 删除经历条目
- 编辑主标题
- 编辑副标题
- 编辑时间区间
- 编辑详细描述
- 子条目显示 / 隐藏

#### 文本类模块
- 编辑模块标题
- 编辑整段文本内容
- 支持普通文本换行
- 支持 HTML 内容输入

### 5.3 排版设置
- 三种版式切换：左对齐 / 居中 / 右对齐
- 简历背景色配置
- 四级字体层级配置：
  - 大标题
  - 一级标题
  - 二级标题
  - 正文
- 每层支持字体、字号、加粗设置
- 支持全局行高调整
- 支持页面边距调整
- 支持模块间距调整

### 5.4 预览能力
- 右侧 A4 纸张实时预览
- 编辑内容实时同步到预览区
- 预览区仅展示，不直接编辑
- 支持头像、列表、文本、HTML 内容渲染

### 5.5 导出能力
- 导出前弹出确认框
- 支持导出高清 PDF
- 导出时自动处理分页辅助元素

### 5.6 智能排版
- 智能分页
- 模块标题靠近页底时自动下推
- 条目跨页时自动整体下推
- 编辑态显示视觉分页间隔
- 支持一键“智能铺满纸张”

### 5.7 状态管理
- 本地自动保存
- 刷新后保留上次编辑状态
- 撤销
- 重做
- 支持快捷键操作

### 5.8 国际化
- 中英文切换
- UI 文案双语支持

## 6. 页面说明

### 6.1 顶部导航
- 产品名称
- 撤销按钮
- 重做按钮
- 中英文切换按钮
- PDF 导出按钮

### 6.2 左侧边栏

#### 排版设置
- 智能铺满纸张
- 模板风格切换
- 背景色设置
- 字体层级设置
- 行高设置
- 页边距设置
- 模块间距设置

#### 模块管理
- 模块列表
- 模块排序
- 模块隐藏
- 模块删除
- 新增经历模块
- 新增文本模块

### 6.3 中间编辑区
- 根据当前选中模块渲染不同编辑器
- 只在一个区域内集中编辑

### 6.4 右侧预览区
- A4 成品视图
- 不直接编辑
- 实时渲染结果

## 7. 非功能需求

- 纯前端运行
- 无数据库依赖
- 无服务器依赖
- 支持现代浏览器
- 交互尽量流畅
- 预览与 PDF 导出保持高一致性

## 8. 当前版本亮点

- 已具备完整的编辑器工作台形态
- 模块体系清晰，便于扩展
- 排版控制较细，适合不同类型简历
- 智能分页和智能铺满能力具有差异化
- 本地保存和撤销重做提升了可用性
